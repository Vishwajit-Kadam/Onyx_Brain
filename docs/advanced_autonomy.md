# Advanced Autonomy

The v0.0.2 advanced autonomy pass adds a bounded orchestration layer for multi-artifact work.

The pipeline is:

1. understand the goal
2. record self-questions, assumptions, limitations, and knowledge gaps
3. create a workspace profile and work contract
4. build a task graph and select a workflow recipe
5. generate deterministic markdown artifacts
6. validate, review quality, revise, and check consistency
7. add cross-links
8. write a final audit and report card
9. export the package with a manifest of file sizes and hashes
10. store a transparent execution trace

Advanced autonomy is still bounded. It does not add an LLM, network access, hidden background workers, unrestricted shell execution, or outside-sandbox writes.

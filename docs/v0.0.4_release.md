v0.0.4 adds executive autonomy, a native `eframe`/`egui` desktop GUI shell, and creative production planning.

## Highlights

- Desktop GUI shell with modern dark frontend assets.
- AppApi boundary for GUI-safe calls into the runtime.
- Consciousness-inspired self-model, attention state, and metacognition reports.
- Creative production studio for cinematic planning packages.
- GUI smoke, creative, and executive benchmarks.

## Safety

Onyx Brain remains bounded. It is not conscious, not AGI, not sentient, and not a real LLM by default. It does not access the network by default, does not run unrestricted shell commands, and does not write outside sandbox/runtime storage.

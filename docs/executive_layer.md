# Executive Layer

The v0.0.4 executive layer is consciousness-inspired, not conscious.

It provides:

- self-model
- attention state
- metacognitive reports
- executive decisions
- bounded executive status

Commands:

```bash
cargo run -- self-model
cargo run -- attention
cargo run -- metacognition "Analyze current autonomous session"
cargo run -- executive-status
```

Executive mode keeps hard safety boundaries: no unrestricted shell, no hidden workers, no network by default, and no outside-sandbox writes.

# Release v0.0.2

`v0.0.2` upgrades Onyx Brain with a bounded autonomous worker engine.

## Included

- `autonomize` command
- `full-bounded` autonomy level
- rule-based goal understanding
- autonomous phase planner
- markdown presentation artifact workflow
- artifact manifest and artifact inspection
- autonomous validation and repair
- session reports
- autonomy score
- autonomy benchmark
- doctor and brain-status autonomy checks
- multi-artifact packs, workspaces, workflow recipes, and export packages
- task graphs, bounded scheduler, report cards, self-questioning, reflection memory, and queue-run
- release-kit, documentation-pack, and learning-pack generators
- launch/startup packs, technical-report packs, and product-spec packs
- work contracts, done definitions, workspace profiles, execution traces, local research briefs, knowledge gaps, final audits, v2 report cards, cross-links, and export manifests

## Safety Expectations

This release remains sandboxed, deterministic, and local by default. It does not add LLM APIs, network access, GPU logic, unsafe Rust, or unrestricted shell execution.

## Limitations

Onyx Brain remains experimental. It is not AGI, not conscious, and not production-ready automation software.

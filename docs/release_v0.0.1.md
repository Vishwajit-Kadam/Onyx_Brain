# Release v0.0.1

`v0.0.1` is the first public experimental release of Onyx Brain.

Internal prototype labels `v0.1` through `v0.9` were private development milestones. Public versioning starts here.

## Scope

This release includes a deterministic Rust runtime for sparse, disk-backed, recoverable agent workflows.

Included:

- virtual neuron/synapse runtime
- indexed memory hierarchy
- project worker
- goal execution
- worker mode
- skill reuse
- habits and caches
- runtime profiling
- action journal
- sessions
- transactional edits
- snapshots and rollback
- doctor and regression checks
- basic and reliability benchmarks

## Known Limitations

- Experimental release.
- Not AGI.
- Not conscious.
- No LLM by default.
- No network access by default.
- Rust project automation is deterministic and template-based.
- Safety checks help, but this is not production-hardening.

## Safety Expectations

Use Onyx Brain with generated sandbox projects first. Review generated files and reports. Do not use it to operate sensitive systems without additional review and controls.

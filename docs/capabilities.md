# Capabilities

Onyx Brain v0.0.3 is an experimental bounded autonomous worker runtime with a deterministic conversational layer.

## Can Do

- Create deterministic markdown artifact packs.
- Create sandboxed Rust project packages.
- Validate, review, revise, and export generated artifacts.
- Write work contracts, done definitions, final audits, report cards, traces, assumptions, limitations, and knowledge-gap logs.
- Run reliability tools such as doctor, regression-check, journal, snapshots, rollback, and benchmark commands.
- Hold deterministic local conversations with chat, modes, conversation memory, and transcripts.
- Produce debate maps, teaching explanations, critiques, plans, architecture outlines, debugger suggestions, and research outlines.
- Launch a native `eframe`/`egui` desktop studio shell through `cargo run -- gui`.
- Create creative production planning packages with timelines, shot lists, sound/VFX notes, and reports.
- Show a consciousness-inspired self-model, attention state, and executive decisions for bounded orchestration.

## Cannot Do

- It does not browse the web by default.
- It does not create binary PPTX files yet.
- It does not run unrestricted shell commands.
- It does not access files outside sandbox/workspace boundaries by default.
- It does not guarantee factual correctness without external verification.
- It is not an LLM, AGI, sentient, or conscious.
- It does not render actual Hollywood-level video or copy protected franchises.

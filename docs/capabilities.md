# Capabilities

Onyx Brain v0.0.2 is an experimental bounded autonomous worker runtime.

## Can Do

- Create deterministic markdown artifact packs.
- Create sandboxed Rust project packages.
- Validate, review, revise, and export generated artifacts.
- Write work contracts, done definitions, final audits, report cards, traces, assumptions, limitations, and knowledge-gap logs.
- Run reliability tools such as doctor, regression-check, journal, snapshots, rollback, and benchmark commands.

## Cannot Do

- It does not browse the web by default.
- It does not create binary PPTX files yet.
- It does not run unrestricted shell commands.
- It does not access files outside sandbox/workspace boundaries by default.
- It does not guarantee factual correctness without external verification.
- It is not AGI and is not conscious.

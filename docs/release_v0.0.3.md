# Release v0.0.3

Onyx Brain v0.0.3 adds a deterministic conversational intelligence layer to the existing bounded autonomous worker runtime.

## Included

- `chat` command for interactive and one-shot local conversations.
- Mode commands for debate, teacher, Socratic, critic, planner, architect, debugger, research-outline, creative, summarizer, safety-review, product-manager, and coach flows.
- Disk-backed conversation sessions and lightweight conversation memory summaries.
- Personality profiles that adjust wording only.
- Transcript viewing and export.
- Prompt library and conversation benchmark.
- Brain-status, inspect, doctor, and regression-check coverage for conversation state.

## Limits

Onyx Brain v0.0.3 is not an LLM, not AGI, not sentient, and not conscious. It does not browse the web by default and does not verify current facts. Research mode creates citation placeholders and verification notes rather than fabricated sources.

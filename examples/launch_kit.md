# Launch Kit Example

```bash
cargo run -- autonomize --level full-bounded "Create a complete startup launch package for Onyx Brain v0.0.2 including pitch deck, speaker notes, landing page copy, FAQ, technical overview, risk register, roadmap, release notes, social posts, demo script, metrics plan, launch checklist, and final export package"
cargo run -- artifact-packs
cargo run -- export-package latest
cargo run -- trace latest
```

The output is a markdown-only launch package inside the sandbox with a work contract, done definition, execution trace, quality review, final audit, report card, and export manifest.
